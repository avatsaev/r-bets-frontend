About

Personal project created by @avatsaev

Frontend is built with Angular 2.0, running on AWS Elastic BeanStalk with an Elastic Load Balancer for autoscaling

Backend is built with Rails 5.0 in API mode, using ActionCable for websockets, running on AWS EC2 Container Service (scaled on 2 Web Docker containers and 1 Worker container for background jobs)

Continuous Integration and automatic deployments are managed by Jenkins
